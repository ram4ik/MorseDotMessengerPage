# Morse Dot Messenger Landing Page

A modern, responsive landing page for the Morse Dot Messenger iOS app - a beautiful application for learning and communicating in Morse code.

## Features

- **Modern Design**: Clean, professional design with smooth animations
- **Responsive Layout**: Optimized for all devices (desktop, tablet, mobile)
- **Interactive Elements**: Animated Morse code demonstration and smooth scrolling
- **App Screenshots**: Showcase of the app's interface using provided screenshots
- **Performance Optimized**: Fast loading with optimized images and CSS

## Structure

```
├── index.html          # Main HTML file
├── styles.css          # CSS styles and responsive design
├── script.js           # JavaScript for interactions and animations
├── README.md           # This documentation
└── *.webp             # App screenshots (460x996bb*.webp)
```

## Sections

1. **Navigation**: Fixed header with smooth scroll navigation
2. **Hero Section**: Eye-catching introduction with animated Morse code
3. **Features**: Six key features of the app with icons and descriptions
4. **Screenshots**: Gallery of app screenshots showing the interface
5. **Download**: Call-to-action section with GitHub and App Store links
6. **Footer**: Simple footer with links and branding

## Key Features Highlighted

- ✨ Easy to Learn - Interactive tutorials for beginners
- 🔒 Secure Messaging - Encrypted communication through Morse code
- ⚡ Real-time Translation - Instant text to Morse conversion
- 🎵 Audio & Visual - Multiple ways to send/receive messages
- 📚 Practice Mode - Built-in learning and progress tracking
- 📱 Cross-Platform - Beautiful iOS interface

## Technologies Used

- **HTML5**: Semantic markup with accessibility in mind
- **CSS3**: Modern styling with Flexbox, Grid, and animations
- **JavaScript**: Interactive features and smooth animations
- **Google Fonts**: Inter font family for clean typography
- **SVG Icons**: Scalable vector icons for crisp display

## Customization

### Colors
The design uses a blue color scheme that can be easily customized:
- Primary: `#3b82f6` (Blue 500)
- Secondary: `#1d4ed8` (Blue 700)
- Background: `#f8fafc` (Slate 50)
- Text: `#1e293b` (Slate 800)

### Content
All text content can be easily modified in the HTML file. The features section is particularly easy to customize with new icons and descriptions.

### Images
Replace the `460x996bb*.webp` files with your own app screenshots. The layout automatically adapts to the number of images provided.

## Deployment

Simply upload all files to your web server or hosting platform. The page is static and requires no server-side processing.

### GitHub Pages
This landing page is perfect for GitHub Pages deployment:
1. Push files to your repository
2. Enable GitHub Pages in repository settings
3. Your landing page will be available at `https://username.github.io/repository-name`
